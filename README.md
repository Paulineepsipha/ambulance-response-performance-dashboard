# Ambulance Emergency Response Performance Dashboard (Victoria)

This Power BI project analyses ambulance emergency response performance across Victorian LGAs, focusing on:

- Incident demand distribution
- Average and weighted response times
- Regional performance gaps

## Key Insights
- High-demand LGAs tend to maintain faster response times
- Regional LGAs experience longer delays despite lower incident volumes
- Majority of LGAs exceed the 15-minute response benchmark

## Tools Used
- Power BI
- DAX Measures
- Data Visualisation & Performance Analytics

## Data Source
Victorian Ambulance Response Data (Open Government Data)
